API Automation

Framework
1. Node.js
2. Supertest : Membuat request http
3. JEST : Unit test
4. Mocha : Runner, test case management
5. Chai : Assertion method (memastikan expected = actual)

How?

- Get HTTP Request
- Post HTTP Request
- Payload / Data Management
- Reporting with mochawesome
- Intergration/Regression
